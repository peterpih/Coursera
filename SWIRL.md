### Swirl Commands

**Loading Swirl package, from inside RStudio:**
```{R}
install.packages("swirl")
library(swirl)
```
**Load the appropriate swirl lessons:**
```{R}
install_from_swirl(...)
install_from_swirl("Exploratory Data Analysis")
install_from_swirl("Statistical Inference")
```
**Run swirl:**
```{R}
> swirl()
```

### Bayes Theorem

- P(A | B) = P(A & B)/ P(B)  probability of A given B

- P(A & B) = P(A | B) * P(B)


- P(B | A)  = P(B & A)/P(A)
            = P(A|B) * P(B) / P(A)

                              P(A) = P(A | B) * P(B) + P(A | ~B) * P(~B)

- P(B | A) = P(A | B) * P(B) / ( P(A | B) * P(B) + P(A | ~B) * P(~B) )
